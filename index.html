VSC

Visual Studio Code (VSC): Editor kode yang ringan dan fleksibel, ideal untuk pengembangan web. VSC menawarkan fitur seperti pemantauan langsung, penyorotan sintaksis, dan integrasi Git, serta dukungan ekstensi yang luas untuk mempermudah penulisan HTML dan CSS.

HTML

HTML (HyperText Markup Language): Bahasa standar untuk membuat dan menyusun halaman web. HTML menggunakan tag untuk menentukan struktur konten seperti teks, gambar, dan tautan, serta elemen-elemen lainnya di halaman web.

CSS

CSS (Cascading Style Sheets): Bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web yang ditulis dalam HTML. CSS memungkinkan pengaturan warna, font, tata letak, dan elemen visual lainnya untuk menciptakan desain web yang menarik dan responsif.
